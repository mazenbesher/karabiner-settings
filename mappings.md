# Switch MAC Keys
```
Original Mac:   fn , ctrl, alt , cmd
New Mac:        cmd, fn  , ctrl, alt
```

# Windows Remote Desktop / VMWare
```
Original Mac:   fn , ctrl, alt, cmd
New Win:       ctrl,   fn, win, alt
``` 

## For Ditto:
```
FN + Capslock == F10
```
